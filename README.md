# 에어사인 프로 — 에어간판 장비·부자재 전문몰 랜딩페이지

20년 현장 경험을 바탕으로 한 에어간판 장비·부자재 전문몰의 랜딩페이지입니다.

## 파일 구성

```
site/
├─ index.html        ← 메인 페이지 (이 파일을 브라우저로 열면 됩니다)
├─ style.css         ← 전역 스타일 (폰트, 리셋, 반응형)
├─ script.js         ← 인터랙션 (FAQ 아코디언)
├─ images/
│  └─ hero-workshop.png   ← 회사소개 섹션 작업 현장 사진
└─ README.md         ← 이 문서
```

## 사용 방법

1. `index.html` 을 더블클릭하면 브라우저에서 바로 열립니다.
2. 웹 호스팅에 올릴 때는 `site/` 폴더 전체를 그대로 업로드하세요.

## 수정 안내

- **글자 내용** : `index.html` 에서 해당 텍스트를 찾아 바꾸면 됩니다.
- **색상 / 폰트 크기** : 각 요소의 `style="..."` 속성에서 직접 수정합니다.
  - 포인트 네이비 `rgb(14, 42, 71)` / 주황 CTA `rgb(241, 90, 34)`
- **전역 스타일·반응형** : `style.css` 에서 관리합니다.
- **FAQ 동작** : `script.js` 에서 관리합니다.
- **사진 교체** : `images/` 폴더의 이미지를 같은 이름으로 교체하면 됩니다.
  - 장비/부자재 카드와 일부 섹션은 회색 사진 자리(placeholder)로 되어 있으니,
    실제 사진으로 교체하실 수 있습니다.

## 참고

- 본 페이지의 투자금·마진·회수 기간 등 수치는 **예시**이며 실제와 다를 수 있습니다.
- 전화번호(1600-0000)와 상호명("에어사인 프로")은 **가칭/예시**입니다. 실제 정보로 교체하세요.
